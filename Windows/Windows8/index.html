Windows 8

Windows 8.1 — операционная система семейства Windows NT производства корпорации Microsoft, следующая по времени выхода за Windows 8 и перед Windows 10. Предназначена для рабочих станций, персональных компьютеров и портативных устройств; версия, предназначенная для решения серверных задач — Windows Server 2012 R2. По сравнению с Windows 8 имеет ряд обновлений и изменений в работе с графическим интерфейсом. Windows 8.1, так же, как и Windows 8, ориентирована на сенсорные ПК, но не исключает возможности использования на классических персональных компьютерах.
